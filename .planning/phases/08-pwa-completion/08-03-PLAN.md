<task type="auto">
  <name>Task 1: useInstallPrompt hook and InstallBanner component</name>
  <files>src/hooks/useInstallPrompt.ts, src/components/pwa/InstallBanner.tsx, src/components/pwa/InstallBanner.module.css</files>
  <action>
    Create `src/hooks/useInstallPrompt.ts` ("use client" not needed in hook file, but must be used from client component):
    - Export `useInstallPrompt` hook
    - State: `deferredPrompt` (BeforeInstallPromptEvent | null), `isInstallable` (boolean)
    - On mount (useEffect):
      1. Check if already in standalone mode: `window.matchMedia("(display-mode: standalone)").matches` — if so, return early (already installed)
      2. Check localStorage for `"pwa-install-dismissed"` — if set and less than 7 days old, return early
      3. Listen for `"beforeinstallprompt"` event on window
      4. When fired: `e.preventDefault()`, store event in `deferredPrompt`, set `isInstallable = true`
    - Cleanup: remove event listener
    - Export function `promptInstall()`: calls `deferredPrompt.prompt()`, awaits `userChoice`, if "accepted" reset state
    - Export function `dismissPrompt()`: sets `isInstallable = false`, stores `{ dismissed: Date.now() }` in localStorage key `"pwa-install-dismissed"`
    - Return: `{ isInstallable, promptInstall, dismissPrompt }`

    Type the BeforeInstallPromptEvent (not in standard lib):
    ```typescript
    interface BeforeInstallPromptEvent extends Event {
      prompt(): Promise<void>;
      userChoice: Promise<{ outcome: "accepted" | "dismissed" }>;
    }
    ```

    Create `src/components/pwa/InstallBanner.tsx` ("use client"):
    - Import `useInstallPrompt` from `@/hooks/useInstallPrompt`
    - Import styles from `./InstallBanner.module.css`
    - If `!isInstallable`, return null (render nothing)
    - Render a fixed-bottom banner:
      - Left side: "Install MultiRotter for quick access"
      - Right side: "Install" button (calls promptInstall) and "X" dismiss button (calls dismissPrompt)
    - Use CSS module classes for styling

    Create `src/components/pwa/InstallBanner.module.css`:
    - .banner: position fixed, bottom 0, left 0, right 0, background #3293CD, color white, padding 12px 16px, display flex, align-items center, justify-content space-between, z-index 1000, font-family Arial sans-serif, font-size 14px
    - .actions: display flex, gap 12px, align-items center
    - .installButton: background white, color #3293CD, border none, padding 8px 16px, font-size 14px, font-weight bold, cursor pointer, border-radius 4px
    - .dismissButton: background transparent, color white, border none, font-size 18px, cursor pointer, padding 4px 8px
  </action>
