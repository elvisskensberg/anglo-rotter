    **Icon type classification function (exported as `getHeadlineIconType`):**
    Create a pure function that maps ForumThread to icon type:
    - "fire": threads where category is "scoops" AND title starts with "Red Alert", "UAV infiltration", "Alerts for launches", "BREAKING", "LIVE", or "Multiple explosions" (these are security_alert-origin threads in the seed data). Also any thread with viewCount > 5000 as a fallback fire indicator.

    - "normal": everything else

    ```typescript
    const iconMap: Record<"normal" | "fire" | "camera", string> = {

    };
    ```

    **HeadlineRow component:**
    - Props: `{ thread: ForumThread; isEven: boolean }`
    - Renders a `<tr>` with backgroundColor from isEven (use inline hex values "#eeeeee" / "#FDFDFD" — jsdom cannot resolve CSS custom properties per Phase 5 decision)
    - 4 `<td>` cells:
